Definição: Processo pelo qual seres humanos interagem com computadores (PCs, smartphones, tablets, eletrodomésticos, etc.).

Objetivo: Criar sistemas computacionais interativos com boa usabilidade e acessibilidade.

Por que estudar IHC?

  • Analisar modelos comportamentais e perfis de experiência do usuário
  • Garantir satisfação do usuário com o sistema
  • Evitar maus projetos de interface

Problemas comuns no desenvolvimento de sistemas:

  • Foco excessivo no sistema e desprezo pela interface
  • Base teórica dos desenvolvedores fraca ou inexistente
  • Processo de desenvolvimento inadequado
  • Ferramentas inadequadas
  • Interação usuário-sistema não avaliada

Diferença entre Interface e Interação:

  • Interface: Contato entre usuário e sistema
  • Interação: Processo de comunicação entre usuário e sistema

O que é Figma? Software de edição gráfica de vetor e prototipagem, baseado principalmente no navegador web, com versões desktop para GNU/Linux, macOS e Windows.

Usos principais: Criação de interfaces gráficas (UI/UX) para sites, apps, etc., com colaboração em tempo real.

  • Acesse: www.figma.com
  • Versões gratuitas e pagas disponíveis
  • Pode inserir imagens de referência ou criar novo projeto

Criação de Layouts

  • F Frame (Region tools) para criar layouts
  • Selecionar formato do dispositivo (Android, iOS, etc.)
  • Zoom: Fn + para aumentar, Fn - para diminuir

Manipulação de Elementos

  • Alt + arrastar: duplicar em qualquer direção
  • Shift + arrastar: mover em ângulos retos
  • Ctrl+G agrupar elementos

Grid e Formas

  • Criar Grid: Layout Grid > Columns (sugestão: 4 colunas com Margin 16)
  • Shape tools para adicionar formas
  • Corner radius para alterar bordas de ângulo reto
  • Effects > Drop Shadow para adicionar sombras
  • Espaçamentos recomendados: múltiplos de 8 (8px, 16px, 24px...)

Tipografia e Cores

  • Ferramenta Text para inserir e configurar textos
  • Adobe Color pode ser usado para seleção de paleta de cores
  • Estabelecer padrões de ícones e cores nas áreas clicáveis

Conceitos Chave em IHC

Engenharia de Software vs IHC

  • Engenharia de Software: Foca em métodos de produção para qualidade de programação e desempenho
  • IHC: Foca em métodos de projeto e avaliação para boa especificação de interface

Ementa da Disciplina

  • Interface, Interação e Affordance
  • Usabilidade, Acessibilidade e Comunicabilidade
  • Arquitetura da Informação
  • Projeto e prototipação (baixa, média e alta fidelidade)
  • Avaliação Heurística e de Comunicabilidade
  • Experiência do Usuário (UX) e do Consumidor
  • Pesquisas com usuário (Qualitativa, Quantitativa, Teste A/B, etc.)
