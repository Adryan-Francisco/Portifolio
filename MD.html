2. Teoria dos Conjuntos

Conjuntos Numéricos

  • Naturais (ℕ): {0, 1, 2, ...}
  • Inteiros (ℤ): {..., -2, -1, 0, 1, 2, ...}
  • Racionais (ℚ): Frações p/q, q ≠ 0.
  • Reais (ℝ): Todos os números na reta real.

Intervalos

  • Fechado: [a, b] = {x ∈ ℝ | a ≤ x ≤ b}
  • Aberto: (a, b) = {x ∈ ℝ | a < x < b}
  • Semiaberto: [a, b) ou (a, b]
  • Infinito: (-∞, b] ou [a, ∞)

3. Teoria dos Grafos

Definições Básicas

  • Grafo: Conjunto de vértices e arestas
  • Grafo Simples: Sem laços ou arestas múltiplas
  • Grafo Completo: Todos os vértices conectados
  • Árvore: Grafo acíclico e conexo
  • Grafo Bipartido: Vértices divididos em dois conjuntos

Árvore de Decisão

Usada em algoritmos de busca como:

  • Busca binária
  • Algoritmos de ordenação
  • Árvores de decisão em machine learning
  • Jogos (como xadrez)

Complexidade: O(log n) para buscas eficientes

Representação de Grafos

Matriz de Adjacência

  • Matriz quadrada n × n
  • 1 se existe aresta, 0 caso contrário
  • Boa para grafos densos

Lista de Adjacência

  • Array de listas
  • Cada vértice aponta para seus vizinhos
  • Eficiente para grafos esparsos
