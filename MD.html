2. Teoria dos Conjuntos

Conjuntos Numéricos

  • Naturais (ℕ): {0, 1, 2, ...}
  • Inteiros (ℤ): {..., -2, -1, 0, 1, 2, ...}
  • Racionais (ℚ): Frações p/q, q ≠ 0.
  • Reais (ℝ): Todos os números na reta real.

Intervalos

  • Fechado: [a, b] = {x ∈ ℝ | a ≤ x ≤ b}
  • Aberto: (a, b) = {x ∈ ℝ | a < x < b}
