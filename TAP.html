CRUD 1-N em .NET com PostgreSQL

Implementação completa de um sistema de cadastro de produtos e fornecedores com relacionamento 1-N, utilizando as melhores práticas de desenvolvimento.

ASP.NET Core MVC Entity Framework Core PostgreSQL Bootstrap

Principais Funcionalidades:

  • Cadastro completo de produtos e fornecedores
  • Relacionamento 1-N entre entidades
  • Validações de dados no cliente e servidor
  • Operações CRUD completas
  • Interface responsiva com Bootstrap

Técnicas Avançadas

Programação Orientada a Objetos

  • Classes e Objetos: Modelagem de entidades do mundo real
  • 4 Pilares da POO: Encapsulamento, Herança, Polimorfismo e Abstração
  • Padrões de Projeto: Singleton, Factory, Repository, etc.
  • Arquitetura Limpa: Separação de responsabilidades
  • TDD: Desenvolvimento Guiado por Testes
  • Versionamento: Git e fluxos de trabalho colaborativo

Estruturas Básicas

  • Condicionais: if-else, switch-case, operador ternário
  • Loops: for, while, do-while, foreach
  • Coleções: Vetores, Matrizes, Listas, Dicionários
  • Manipulação de Strings: Métodos e expressões regulares
  • Tratamento de Exceções: try-catch-finally
